# Лабораторные работы по Объектно-Ориентированному Программированию

Репозиторий с лабораторными работами по курсу **объектно-ориентированного программирования**.

## О проекте

Это коллекция учебных лабораторных работ, демонстрирующих основные принципы ООП: инкапсуляцию, наследование, полиморфизм и абстракцию. Пока здесь только одна работа (остальные будут добавляться по мере прохождения курса).
